# Bedrock Index

This repository supports Bedrock Index, Bedrock Health Group's public resource library for governance, compliance, provider operations, credentialing, billing readiness, privacy, security, and healthcare regulatory support.

## Purpose

This repository stores public-facing policy records, public governance materials, templates, and site infrastructure for Bedrock Index.

## Important

This repository is for public-facing Bedrock Health Group resources and public policy records.

Internal-only policies, procedures, playbooks, administrative instructions, security procedures, and sensitive operational materials are maintained separately in Bedrock Health Group's private internal policy environment.

## Core Principles

- Public materials should be accurate, current, and appropriate for external publication.
- No policy is considered approved unless it has gone through the required review and approval process.
- Policy and resource content should be written, reviewed, and approved by appropriate business, compliance, and legal reviewers.
- The repository structure supports version control, document control, review workflow, and public publication through Bedrock Index.
