# Start Here

Use this section to understand where to begin in Bedrock Index.

Bedrock Index is organized around provider tasks, not just regulations.

## Common Starting Points

| If you need to... | Start with... |
|---|---|
| Understand whether your organization is ready | Provider Readiness |
| Prepare for credentialing or payer enrollment | Credentialing & Enrollment |
| Learn about Medicare or Medicaid enrollment | Medicare & Medicaid |
| Prepare billing operations or reduce denials | Billing & Revenue Cycle |
| Review public compliance readiness concepts | Compliance Readiness |
| Find public Bedrock policies | Policy Library |

## Public Information Notice

Bedrock Index provides public educational resources and public policy records. It does not replace payer instructions, CMS requirements, state Medicaid requirements, legal advice, billing advice, or provider-specific compliance guidance.
