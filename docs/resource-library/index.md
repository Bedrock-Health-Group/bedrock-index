# Resource Library

This section will organize public Bedrock Index resources, checklists, explainers, and reference pages.

## Planned Resource Types

| Resource Type | Purpose |
|---|---|
| Checklists | Help users assess readiness. |
| Explainers | Translate complex requirements into plain language. |
| Resource directories | Point users to official external resources. |
| Comparison guides | Help users understand related concepts. |
| Public templates | Provide standardized public reference formats. |

## Intended Use

This section is the public resource hub for Bedrock Index materials that are not formal Bedrock policies.
