# Credentialing & Enrollment

This section will organize public resources related to provider credentialing, payer enrollment, and enrollment readiness.

## Planned Resources

| Resource | Purpose |
|---|---|
| Credentialing Overview | Explains the credentialing process in plain language. |
| Provider Enrollment Overview | Explains common enrollment steps. |
| NPI and Taxonomy Basics | Helps users understand provider identifiers and taxonomy codes. |
| Payer Credentialing Checklist | Lists common payer credentialing preparation items. |
| Common Credentialing Issues | Summarizes common gaps and delays. |

## Intended Use

This section helps users understand the general credentialing and enrollment process. Users should verify requirements with payers, CMS, state Medicaid agencies, and other applicable authorities.
