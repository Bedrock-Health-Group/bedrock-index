# Bedrock Index

Bedrock Index is Bedrock Health Group's public resource library for governance, compliance, provider operations, credentialing, billing readiness, privacy, security, and healthcare regulatory support.

This site provides public-facing policy records and structured resources to help providers, partners, and stakeholders understand Bedrock Health Group's governance standards and public compliance resources.

## Current Public Library Areas

- Policy Library
- Governance
- Privacy & Security

Additional Bedrock Index resource areas may be added over time, including provider regulations, credentialing, Medicare and Medicaid resources, billing and revenue cycle guidance, and accessibility resources.

## Public Information Notice

This site is intended for public-facing information and public policy records.

Internal-only policies, procedures, playbooks, administrative instructions, security procedures, and sensitive operational materials are maintained separately in Bedrock Health Group's private internal policy environment.

## Source of Record

The live page in Bedrock Index is the controlled public record for public materials published here.

Printed, downloaded, or PDF copies should be verified against the live Bedrock Index page before being relied upon.
